## 2. 討伐ドロップ（巻物のみ）

討伐ドロップで出るのは **巻物のみ**。バフ薬は宝箱（§3）から。

1. **ドロップ判定**: 討伐ごとに `SCROLL_DROP_CHANCE = 0.67%` でドロップ有無を抽選。
2. **品目抽選**: 当たったら、巻物3種を均等（重み各1）にランダム抽選。

| アイテム | 計算 | 実効率 | 目安（1個出るまで） |
|---|---|---|---|
| 力の巻物   | 0.67% × 1/3 | **約 0.22%** | 約 450 討伐 |
| 堅の巻物   | 0.67% × 1/3 | **約 0.22%** | 約 450 討伐 |
| 生命の巻物 | 0.67% × 1/3 | **約 0.22%** | 約 450 討伐 |
| 巻物いずれか | 0.67% | 約 0.67% | 約 150 討伐 |

> 階層・敵種別によるドロップ率の変動はなし（一律 0.67% 判定）。

## 3. 宝箱（バフ薬の入手元）

狩場のフィールドに湧く宝箱を **[E]キー** で開けると、**バフ薬1個＋ゴールド**を入手。

| 項目 | 値 |
|---|---|
| 出現場所 | 狩場のみ（町には出ない） |
| 同時出現数 | 常時 **2個**（`TREASURE_TARGET`） |
| 補充間隔 | 約 **12秒**（`TREASURE_SPAWN_COOLDOWN_MS`） |
| 開封 | 近接（48px以内）で `[E]`。サーバー権威で距離検証 |
| 中身（バフ薬） | 力の薬 / 俊足の薬 を **均等ランダムで1個** |
| 中身（ゴールド） | `30 + 階層×20 + 0〜19`（深層ほど多い） |

> 宝箱は誰でも開封可（ゴールド／バフ薬は有限供給ではない）。開封時のバフ薬・ゴールドはログイン者のみ永続化。

## 4. ゴールド獲得（購入の元手）

討伐ごとに獲得。出典: `onMobKilled`。

```
獲得G = max(1, round((2 + floor × 2) × 敵種別のexpMul))
```

- `floor` = 現在の階層（町=0、狩場 B1〜B5＝1〜5）。深層ほど多い。
- `expMul` = 敵種別ごとの倍率（ボス等は高い）。
- このほか宝箱開封でも入手（§3）。

## 5. 入手・消費まとめ

| アイテム | 入手 | 消費/使用 |
|---|---|---|
| 回復薬 / 上級回復薬 | 町ショップで購入（🪙30 / 🪙80） | 使用でHP回復（満タン時は消費しない） |
| 力の薬 / 俊足の薬 | **宝箱**（狩場・[E]で開封） | 使用で30秒バフ |
| 力/堅/生命の巻物 | 討伐ドロップ | 使用で恒久ステータスアップ（アイテムタブからクリック使用） |
| 全アイテム | — | 町ショップ「売却」タブでゴールド化 |

> 巻物は誤爆防止のためホットキー（数字1〜4）対象外。アイテムタブからクリックで使用。
